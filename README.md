# RESUME-AI-PRO

A Flask-based web app that analyzes resumes, highlights missing skills, suggests a learning roadmap, and prepares interview questions for a target role.

## Features
- Resume text input or PDF/DOCX upload
- AI-powered skill gap analysis
- Personalized roadmap suggestions
- Interview question generation
- Modern landing page and dashboard UI

## Setup
1. Create and activate a virtual environment
2. Install dependencies:
   `ash
   pip install -r requirements.txt
   `
3. Run the app:
   `ash
   python app.py
   `
